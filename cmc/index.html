Course Module Catalog extractor

Getting the modules ready for the PGT course pages looked fiddly so I've tried to speed up the process.

What does it do?

On a module page it plucks out the information we want to add to the main site and prepares the:

  • Component name
  • Accordion title
  • Module description

It shows the information in a modal, ready for you to copy into the CMS.

If you run this on a course page, it will remove modules with no credits.

If you run this on other pages or sites you may get a prompt for course search. Other sites will stop my code from running on their pages so you may see nothing. So this script shouldn't do anything on other sites.

Drag the following link to your bookmark bar, and click it when you're on a Course Module Catalogue page.
